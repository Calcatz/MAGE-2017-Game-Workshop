# MAGE-2017-Game-Workshop
This is the finished project of MAGE 2017 Game Workshop about making an FPS Multiplayer game using Unity Multiplayer.

This game features:
-Character Movement (FPS)
-Attack (Shooting)
-Character Animation
-HUD System
-Capture Point System
-Multiplayer System
-Respawn
-Adding Music
-Win-Lose State
-Main Menu Scene
